# 🍔 Food Delivery Time Prediction Using Machine Learning

An end-to-end Machine Learning project that predicts **food delivery time** based on delivery distance, preparation time, courier experience, weather, traffic conditions, time of day, and vehicle type.

The project includes a trained **Polynomial Regression** model and a modern **Streamlit** web application for real-time predictions.

# 📊 Dataset Features

The model predicts delivery time using the following inputs:

| Feature | Description |
|---------|-------------|
| Order ID | Unique Order Identifier |
| Distance (km) | Delivery Distance |
| Preparation Time | Restaurant Preparation Time |
| Courier Experience | Delivery Partner Experience |
| Weather | Weather Condition |
| Traffic Level | Traffic Density |
| Time of Day | Morning / Afternoon / Evening / Night |
| Vehicle Type | Bike / Scooter / Car |

# ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Raj-Rathod-Ai/Food_Delivery_Time-Using-ML.git
```

### Navigate

```bash
cd Food_Delivery_Time-Using-ML
```

### Install Dependencies


